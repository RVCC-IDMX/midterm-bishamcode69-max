# IDMX 225 Midterm Project: Bisham Portfolio

## Overview
This is a fully responsive Web Developer portfolio built as a midterm assessment. The site showcases featured projects, client testimonials, and includes a prominent call-to-action (CTA) for resume download.

## Design & Aesthetics
- **Visual Theme:** Modern Dark Mode, utilizing a custom green accent (`#22c55e`) based on the provided Tailwind configuration.
- **Responsiveness:** Features three distinct media states (Mobile, Tablet @768px, Desktop @1000px) ensuring cross-device readability and layout quality.
- **Creativity:** Includes custom hover effects on project/review cards and a unique offset styling on the Hero image for a professional, high-effort aesthetic.

## File Structure
- `index.html`: Main structure and content.
- `css/style.css`: All custom styling and media queries.
- `img/`: Contains all required project images and the hero image (`profile.jpg`).
